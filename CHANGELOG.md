# Changelog:

## Version 0.1

### Start Menu

- The search bar takes up the entire width of the Start menu
- Added a "Recommended" section (Settings -> Personalization -> Start menu -> enable/disable "Show recently added apps")
- The "Pinned" and "Recomended" heading has been removed
- The open folder from "Pinned" has been moved to the left
- The "Phone Link" and the "User Panel" have been moved lower
- Added a hover effect similar to the TranslucentStartMenu Theme
- Slightly increased the opacity of the dark gray background for all sections
          
### Taskbar

- Slightly increased the opacity of the System Tray background
          
### Notifications Center

- Decreased the opacity of the "Control Center" background 
- Removed the notification block above the calendar
- Removed the "Focus" block below the calendar
- Changed the background opacity for better readability
