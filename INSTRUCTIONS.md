# Instructions

1. Copy the `resources\w11-grey.png` file into `C:\Resources\Icons\Start Orbs`. If the directory does not exist, create it.
2. Copy and paste the contents of the `Start Menu >> Windows Glass.json` file into `WindHawk >> Windows 11 Start Menu Styler >> Advanced >> Mod settings`
3. Copy and paste the contents of the `Taskbar >> Windows Glass.json` file into `WindHawk >> Windows 11 Taskbar Styler >> Advanced >> Mod settings`
4. Copy and paste the contents of the `Notification Center >> Windows Glass.json` file into `WindHawk >> Windows 11 Notification Center Styler >> Advanced >> Mod settings`
